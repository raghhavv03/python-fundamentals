# Python Fundamentals

This repository contains my Python learning journey and practice exercises.

## Topics Covered

- Variables and Data Types
- Conditional Statements
- Loops
- Functions
- Lists, Tuples, Sets, Dictionaries
- OOP (Classes, Inheritance, Polymorphism)
- File Handling
- JSON
- Exception Handling
- List Comprehensions

## Repository Structure

```
python-fundamentals/
├── lessons/          # Example scripts for each part (part_1.py – part_5.py)
├── assignments/      # Assignment solutions and PDFs (part_*_assignment.py / .pdf)
├── notes/            # Lecture notes PDFs (part_*_notes.pdf)
└── data/             # Data files used by the scripts (JSON, TXT)
```

Run scripts from the repo root so data paths resolve correctly, e.g.:

```bash
python lessons/part_5.py
```

## Purpose

To build a strong foundation in Python programming through hands-on practice and problem solving.
